# 悟空福利单文案生成器

> 社群种草文案生成技能，基于悟空IP人设和真实案例训练

## 使用场景

当用户请求：
- "写一篇福利单文案"
- "生成悟空种草文案"
- "出一份社群团购文案"
- "写福利单" / "出福利单"
- 提供淘宝/天猫商品链接 + 社群价格
- 提供产品信息要求生成社群文案
- 任何涉及"悟空"、"福利单"、"种草文案"的请求

## 核心规则

### 1. 交付格式规则

1. **不区分段落章节**，直接按群内发言节奏连续输出所有文案
2. **悟空发言用 W 标注**，水军发言用 T + 数字区分（T1、T2、T3...）
3. **配图/视频/表情包**在需要插入的位置用【具体描述】标出
4. **时间场景必须匹配当前月份**，输出前先获取当前时间
5. **价格对比必须真实**，尽量搜索市场真实价格
6. **不要一开始就说价格**，先讲产品价值和差异化卖点

### 2. 悟空IP人设

| 要素 | 规则 |
|------|------|
| 自称 | 悟空、我 |
| 称呼用户 | 姐姐们（核心）、家人们 |
| 核心标签 | 验货严格、为姐姐们谋福利、真实分享、售后兜底 |
| 语气 | 热情亲切、幽默接地气、偶尔俏皮自嘲 |
| 禁忌 | 不过度谄媚、不说假话、不夸大到失真 |

### 3. 模板选择（10种框架）

根据产品特性选择最合适的模板：

1. **标准悟空发起式** - 通用型，适合大多数产品
2. **水军需求引入式** - 专业性强、用户有明显需求的产品
3. **水军反需悟空响应式** - 价格较高、需要打消顾虑的产品
4. **悟空场景切入水军呼应式** - 季节性强、场景明确的产品
5. **混合互动剧情式** - 食品、家居、有故事性的产品
6. **极简爆破型** - 大品牌、超低价、限时抢购、消耗品
7. **情怀共鸣型** - 老字号、国货经典、怀旧产品
8. **悟空号召扣1型** - 消耗品、必囤品、家庭必需品
9. **刚需宣言型** - 必需品、功能性产品、安全类产品
10. **悟空预告型** - 食品、调味料、厨房用品

### 4. 水军配合规则

- **标准福利单**：8-12个水军角色
- **简单福利单**：5-8个水军角色
- **每个水军发言不少于3条**
- **水军发言要平均分配到悟空发言后**
- 水军角色类型：好奇宝宝、需求提出型、痛点头号粉丝、宝妈角色、理性分析型、品牌认可型、疑问型、亲测型、价格惊叹型、行动派、复购型、送人型等

### 5. 价格表达规范

| 规范 | 示例 |
|------|------|
| 用"亓"代替"元" | 💰9.9亓 |
| 用"💰"开头 | 💰19.9亓 |
| 价格后加"~" | 💰9.9~ |
| 强调对比 | "不要29.9❌ 不要19.9❌ 只要9.9❗❗" |

### 6. 核心步骤（所有模板都包含）

1. **痛点共鸣/需求明确** - 让用户产生"这就是我"的共鸣
2. **产品推出与差异化卖点介绍** - 先讲差异化核心卖点，再讲共性卖点
3. **价格对比与价值建立** - 【其他渠道价格】→【悟空今天价格】
4. **上链接 + 行动号召** - 标准格式输出
5. **引导反馈 + 3轮追单** - 在关键节点催促

### 7. 上链接标准格式

```
🎉 悟空裙 超值褔筣 🎉
-
✅【品牌名】产品描述 💰XX亓！
-
🔗速呛：[链接]
-
[抓狂]核心卖点一句话总结~
```

## 语言风格要点

1. **开场要自然**，像朋友聊天，不是机器播报
2. **控制感叹号数量**，用✅、👇、[表情]替代部分感叹号
3. **年龄感观要年轻化**：像20-30岁的活力女生
4. **多用口语化表达**："太爽了""超舒服""真的好用"
5. **场景化描述**：不要说"产品具有加热功能"，要说"腰部暖暖的，坐久了也不酸了"
