# 悟空福利单文案生成器 - 员工配置指南

## 📋 说明

本指南帮助你配置 QClaw 钉钉机器人，使其能够自动使用公司统一调试的最新版本技能包。

---

## 🚀 配置步骤

### **第1步：修改 QClaw 配置文件**

找到你的 QClaw 配置文件（通常在以下位置之一）：
- Mac: `~/.qclaw/openclaw.json`
- Windows: `C:\Users\<你的用户名>\.qclaw\openclaw.json`

### **第2步：添加技能包地址**

在配置文件中找到 `skills.load.extraDirs` 部分，添加以下地址：

```json
{
  "skills": {
    "load": {
      "extraDirs": [
        "https://github.com/1061114325-droid/-wukong-fuli-copy.git"
      ]
    }
  }
}
```

**注意：** 如果已有其他技能包地址，保留原有的，只添加新的一行。

### **第3步：重启 QClaw**

修改配置后，重启 QClaw 使配置生效：
- Mac: 退出 QClaw 应用后重新打开
- Windows: 关闭 QClaw 后重新启动

### **第4步：验证配置**

重启后，在钉钉中测试机器人是否正常工作：
- 发送消息：`写一篇福利单文案`
- 如果机器人正常回复，说明配置成功

---

## 🔄 自动更新机制

配置完成后，你的机器人会自动：
- 每 5 分钟检查 GitHub 是否有新版本
- 如果有新版本，自动下载并加载
- 无需手动操作

当管理员在本地调试并推送新版本后，你的机器人会在 5 分钟内自动更新。

---

## ❓ 常见问题

**Q: 我的机器人没有自动更新怎么办？**
A: 检查网络连接，确保能访问 GitHub。如果仍然不行，手动重启 QClaw。

**Q: 配置后机器人不工作了？**
A: 检查配置文件格式是否正确（JSON 格式），确保没有语法错误。

**Q: 如何知道当前使用的是哪个版本？**
A: 在钉钉中询问机器人当前版本信息。
